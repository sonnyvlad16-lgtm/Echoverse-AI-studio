# EchoverseAI Studio

Platforma all-in-one de inteligență artificială pentru generare imagini și text.

## 🚀 Proiect în dezvoltare

### Stack-ul proiectului:
- **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla ES6+)
- **Backend:** Node.js + Express (sau Python + FastAPI)
- **Bază de date:** PostgreSQL
- **API-uri AI:** OpenAI (GPT, DALL-E), Replicate
- **Plăți:** Stripe
- **Hosting:** Vercel (frontend), Railway/Render (backend)

## 🔧 Instalare și Setup

### Frontend
```bash
cd frontend
npm install
```

### Backend
```bash
cd backend
npm install
cp .env.example .env
```
